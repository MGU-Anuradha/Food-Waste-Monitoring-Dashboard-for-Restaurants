# Food Waste Monitoring Dashboard for Restaurants

A real-world Power BI analytics project designed for undergraduate students to experience the complete Business Intelligence (BI) lifecycle — from raw operational data to actionable business insights.

This project focuses on helping restaurant chains monitor, analyze, and reduce food waste using Power BI dashboards and data analytics techniques.

---

# Project Objective

The objective of this project is to build an interactive Food Waste Monitoring Dashboard that enables restaurant management to:

- Identify the most wasted food items
- Detect inefficient restaurant branches
- Understand root causes of food waste
- Analyze waste trends over time
- Estimate profit loss caused by waste
- Predict future waste patterns
- Support data-driven operational decisions

---

# Business Scenario

A restaurant chain operating across multiple branches is experiencing increasing food waste, leading to:

- Higher operational costs
- Reduced profitability
- Inventory inefficiencies
- Poor kitchen planning

Currently, waste tracking is manually maintained using Excel spreadsheets, making it difficult for management to gain meaningful insights.

The organization needs a centralized analytics solution to answer key business questions such as:

- Which food items generate the highest waste?
- Which branches are underperforming?
- What are the main reasons for waste?
- Which shifts or days contribute most to waste?
- How can future waste be reduced?

# Dashboard Pages

1. Total Waste Quantity
